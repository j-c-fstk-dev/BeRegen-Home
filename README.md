**BeRegen** is a non-profit foundation dedicated to building and stewarding open-source infrastructure for the regenerative economy. Our mandate is to research, develop, and maintain public goods that empower individuals and communities to create positive environmental and social impact.

This repository contains the source code for the main web portal, which serves as the central gateway to the entire BeRegen ecosystem.

## Core Principles

Our work is guided by a strong ethical compass to ensure our actions align with our regenerative mission.

## Technology Stack

The BeRegen Portal is built with a modern, performant, and scalable tech stack:
- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **UI**: [React](https://react.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [ShadCN UI](https://ui.shadcn.com/)
- **AI/Generative**: [Genkit](https://firebase.google.com/docs/genkit)
- **Deployment**: Firebase App Hosting

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v18 or later)
- npm or yarn

### Installation

1. Clone the repo:
   ```sh
   git clone https://github.com/your_username/beregen-portal.git
   ```
2. Navigate to the project directory:
   ```sh
   cd beregen-portal
   ```
3. Install NPM packages:
   ```sh
   npm install
   ```
4. Run the development server:
   ```sh
   npm run dev
   ```
Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

## Future Implementations

We have an ambitious roadmap to expand the BeRegen ecosystem. Here are some of the features and platforms we plan to build next:

- **On-chain Governance Integration**: Fully integrate the RegenFund DAO with the portal, allowing for on-chain proposal submission, voting, and treasury management.
- **Hypercerts for Impact**: Implement Hypercerts within RegenHub to create verifiable, tradable certificates of impact for completed projects.
- **AI-Powered Curation**: Enhance RegenHub with advanced AI models to assist human curators in validating and verifying submitted actions, increasing scalability and reducing bias.
- **Personalized Learning Paths**: Develop AI-driven recommendations on RegenFlix to guide users through personalized educational journeys based on their interests and goals.
- **Mobile App Enhancements**: Launch new modules in the BeRegen App for community gardening, local exchange, and habit-tracking gamification.
- **Cross-Ecosystem Reputation**: Implement the `$LIFE` soulbound token to create a unified reputation system that reflects contributions across all BeRegen platforms.
