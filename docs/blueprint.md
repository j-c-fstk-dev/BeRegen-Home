# **App Name**: BeRegen Portal

## Core Features:

- Homepage with Ecosystem Map: Interactive ecosystem map showcasing BeRegen's various components: App, Hub, Fund, Flix, and Store.
- Landing Pages: Individual landing pages for each core component: Foundation, App, Hub, Fund, Flix, and Store, detailing their purpose and features.
- Community Support Channel: Integration with Patreon and Giveth to allow users to support the BeRegen ecosystem directly.
- Transparency Dashboard: Display on-chain and off-chain metrics via TheGraph/Dune embeds to ensure transparency and accountability.
- Updates Section: A blog-style section, possibly integrated with Beehiiv, to post updates, news, and newsletters related to BeRegen.
- Documentation Hub: Docusaurus-based documentation for technical details, whitepapers, and governance handbooks.
- Curator/Validator application form: Application forms for potential curators and validators with an interface to approve user generated content, and ensure trustworthiness of submitted content and data.

## Style Guidelines:

- Primary color: Vivid Green (#50C878) to represent growth, regeneration, and connection to nature.
- Background color: Desaturated Light Green (#E8F4E7) to provide a calm and natural feel.
- Accent color: Cyan (#4ECDC4) to highlight interactive elements and add a modern touch, complementing the green tones.
- Headline font: 'Space Grotesk' (sans-serif) for a modern and tech-forward feel, particularly for headlines.
- Body font: 'Inter' (sans-serif) to ensure readability and a clean, modern aesthetic throughout the body text.  Paired with Space Grotesk
- Use clean, minimalist icons to represent different aspects of the BeRegen ecosystem and its various features.
- Implement a responsive, grid-based layout with generous spacing (p-6+) to create a sense of openness and clarity. Corners should have a 2xl border radius.
- Subtle animations and micro-interactions using Framer Motion, such as fade-in effects, hover glows, and smooth scrolling.
